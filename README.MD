## Setting Up App
To run this app you will need to get some properties in .enviroment (.env), these properties will be the next ones

```sh
# Mongodb
ATLAS_URI=<YOUR URL FROM MONGODB>
DB_NAME=<YOUR DATABASE NAME>
```
